About

Subject is a free and open platform for open information, knowledge and sharing. Content created with Subject is publicly visible and available to everyone on the internet. Everyone is free to browse the content hosted on Subject but to create content users need to be registered. The motivation for creating yet another wiki/content management system came from the fact that other applications seems to focus more on additional services than on the actual content.

Features

To see the implemented and planned features for Subject, visit https://subject.se/space/syops-corner/subject-features

Usage

Built upon the concept of Spaces and Pages, anyone can create and organize hierarchical information. Content is created with the Markdown markup language or plain text. Contributed content is available at https://www.subject.se/space/space-name/page-name.

Example

Let's take a look at the fictional computer club called Cosmos. Cosmos use Subject to publish and share information about their club, activities and projects.

  • Cosmos Computer Club (this is a Space)
    • About (This is a Page)
      • History (These are child Pages to the parent Page "About")
      • Goal
      • Members
        • Active (These are child Pages to the parent Page "Members")
        • Inactive
    • Events
      • Upcoming events
      • Event archive
    • Projects
      • Software
      • Hardware

By structuring information this way, we can represent everything from cookie recipes, software development project documentation, poetry to non-profit organizations and collections.
